# Python Boshlovchilar Uchun Mini Kurs

Bu repository Python dasturlash tilini endi boshlayotganlar uchun mo'ljallangan.  
Har bir dars **kod + izoh + misollar** bilan tushuntirilgan.

## Darslar ro'yxati

1. **01_print.py** – `print()` funksiyasi
2. **02_input.py** – `input()` bilan foydalanuvchi ma'lumotini olish
3. **03_if_else.py** – `if`, `else`, `elif` shart operatorlari
4. **04_for_loop.py** – `for` sikli
5. **05_while_loop.py** – `while` sikli, `break`, `continue`
6. **06_function.py** – funksiyalar, parametrlar, `*args`, `**kwargs`
7. **07_list.py** – listlar, element qo‘shish, o‘chirish, indekslash
8. **08_dictionary.py** – dictionary, kalit-qiymat, `.items()`, `.keys()`, `.values()`
9. **09_tuple.py** – tuple, indekslash, `.index()`, `.count()`
10. **10_args_kwargs.py** – `*args` va `**kwargs` misollar bilan
